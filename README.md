# Full-Stack Developers Coding Exercise
This is is full-stack developers coding exercise built using .NET Core and React technologies.

A database schema is created with the following specification:

<img src="https://github.com/Rapheta/FullStack-Developers-Coding-Exercise/blob/main/imgs/database.png" width="60%"/>

The application has been developed through an web interface and covers the following functionality:

<ul>
  <li>List the records of each of the entities</li>
  <li>Modify each of the entity records</li>
  <li>Register new records of a specific entity</li>
</ul>
